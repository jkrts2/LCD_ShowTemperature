# LCD_ShowTemperature
 单片机学习代码备份
# 使用模块
基于DS8B20模块读取温度，LCD1602模块显示温度
# 基本介绍
LCD1602文件为LCD驱动，OneWire文件实现DS8B20基本单总线通讯，DS18B20文件实现模块初始化，温度数据传输功能
